# PickHacks2026-OASISProject
36 Hour deadline group project with the goal of creating an optimization model that an determine optimal reservoir placement in a grid to minimize water transportation costs.

In order to run the following program, download the full repository or clone it via git using your preferred method. Once downloaded, in a terminal, run the following command "pip install -r requirements.txt", preferably in a virtual environment, in order to install our project's dependencies. In order to run the program, select "run" in your preferred IDE and go to the link provided by flask in the terminal to view the frontend.
